# HMCet3
We present a non-perturbative lattice study providing numerical evidence for a mechanism by which M-theory compactified on a G2-holonomy manifold with codimension-7 conical singularities selects a discrete infrared attractor—labelled by the intersection triple (n1,n2,n3) = (1,207,17)— without recourse to anthropic or continuous fine-tuning.
